# 我判断出了肠绞痛，但宝妈的追问、让我无语

当时，这家娃快满3个月了，
连续几天，都在傍晚
开始痛苦地哭闹。
一哭就将近1小时，
断断续续到晚上才平静。
阿姨打电话给我，
怀疑是肠绞痛，因为之前
她有过类似的经历。
当我和阿姨一起，
按SADE一项项排查后，
我也基本认同她的判断。
(详见插图)
当我把自己的判断，
笃定地告诉家长后，
宝妈边给宝宝擦汗，
边疑惑问我：
为什么之前是好好的，
现在得了肠绞痛病呢？
我还没有解释完：
“这不是病……”
奶奶拿着奶瓶，一进屋，
就急切地插嘴道：
“现在怎么办？
怎么让宝宝不哭？
哭伤了怎么办？
实在太可怜了……”
这时，在一旁拖地的阿姨
赶紧抬头，轻轻说：
“我刚刚特地给宝宝
做了排气操，
但宝宝在哭，不配合，
我就换飞机抱拍她，
昨天我也教你排气操动作……”
边说边看向宝妈。
宝妈接过奶瓶喂水，
“阿姨，我没说你不做事，
但是，没用啊！
孩子不还是哭不停？
好像，还更严重了。”
阿姨拖完地、出了房间，
我接过宝妈的话头：
“阿姨的做法，是对的，
目前的状况，也基本正常，
确实没有，更好的办法。
另外，宝宝在哭，别喂水……”
我还没说完，
奶奶就不乐意了，
“别跟我说喂水，我就问你，
这么小的孩子，
哭哑了嗓子，正常吗？
签合同时，
你不是说很专业的吗？
阿姨很有经验的吗？
哄哭都不会，还正常……”
接着是，翻来覆去的车轱辘话，
接着是，我见缝插针反复解释。
几番来回后，快21点了，
阿姨用温水，给宝宝抹了澡，
宝宝也渐渐安静睡去。
因为拿不出，有效的解决方案，
家长的脸上，写满了——
“很不满意”和“非常失望”，
我只能匆匆告辞、郁郁而回。
3天后，客户找了个理由
辞退了阿姨，很遗憾。
因为我们事后复盘，
阿姨的作业，
可圈可点(详见插图)。
我能理解，家长那时无心了解，
关于肠绞痛的“来龙去脉”，
我也能理解，看着孩子肠绞，
家长，揪心揪肺的感觉，
我更能理解，当家长得知：
无药可治、只能硬熬，
这时的无助和崩溃。
但我不能像关开关一样，
让哭声立刻停止。
只能默默期盼——
宝宝尽快渡过这段生长期。
从那以后，
我会向客户聊聊——
“肠绞痛”的常识A、B、C，
提前建立预期，
只为不被再次追问。
因为有些哭闹，
不能立刻平抚。
因为有些阶段，
要大人陪着孩子，
一点点熬过去。
