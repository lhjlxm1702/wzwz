# 半年心血归零，我黯然关掉电脑

去年，临近元旦的一个晚上，
终于敲打出来了——
宝宝哭声翻译器的雏形。

我抓起手机，第一次正式测试。

糟糕，怎么是报错⚠
再测~~报错⚠
反复几次~~始终报错⚠

我心里一颤：没道理啊？

翻来覆去，检查到半夜，
才发现，所有后端数据，
被纯小白的我，
不经意间，放到了国外服务器。

所以，国内的手机，
打不开这款工具。

这，就意味着——
当我准备入住新房时，才突然发现：
刚刚装修完的，是别人家，
自己的毛坯房，在隔壁。

当时，脑袋一片空白🤯，
手心渗出细汗，却浑身寒意；

我推开记录本、关掉电脑、熄了台灯，
蜷缩在藤椅里。

冰冷的冬雨，拍着玻璃窗，
满脑尽是，报错的“黄色三角”。

闭上眼睛，又想着——

敲出的3000行代码、
编写的130页备忘录、
翻烂的2本教材……

似乎都在笑话
我这个野生码农。

现在…怎么办❓
就此放弃？淘宝外包？从零开始？

连着几天，
整个人恍恍惚惚、话也少了。

一天晚餐后，老婆收拾碗筷时，
调侃我：最近更年期了吗？

当她知道原委后，
竟没心没肺地逗我：
“反正就在隔壁，那就蚂蚁搬家呗”🐜

正要反驳她：搬家、不是、复制粘贴，
微信弹出提示——
收到一段，月嫂新录的宝宝哭声。

坐在干净的餐桌前，
我轻轻地、划看6000多段录音，
偶尔点开一段听听。

每段背后，都有一个平凡的故事，
每段里面，都暗藏破解哭声的密码，

他们在我手机里，
静静地，等了5年；

再等5年吗？我都退休了……

那晚，听了很久很久的录音，
我重新坐回、21吋的大屏幕前🖥️。

看了看窗外的银行霓虹灯，
硬着头皮，点开了“阿里云"……

全部完成“蚂蚁搬家”的那天，
我推开窗，
风中已没有了寒意，

南昌，已经进入
春暖花开的3月份了。

接着，放到小群测试，
竟收到老客户们、近百条反馈，

现在，我的“婴语翻译”工具，
不仅脱胎换骨，
而且完成了雏形的蜕变。
